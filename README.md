# WxMovie

This page shows all 10 random movies filtered by year and genre.<br/>
Live demo: https://wxmovie.netlify.app

## Start the project

```
git clone <repo_url>
cd <repo>
npm install
npm start
```

## Technology

- Frontend:
  - ReactJS
  - TypeScript
  - TailwindCSS
- Backend:
  - Supabase
- Hosting:
  - Netlify
